Image Colour Contrast Checker

Drop in images to check WCAG AA / AAA text-contrast. Runs entirely in your browser — nothing is uploaded.

Privacy Statement

This tool is designed to be 100% private by architecture. Here is exactly what happens — and does not happen — when you use it.

Your images never leave your device

Every step of the analysis — image decoding, OCR text detection, colour sampling, and contrast calculation — runs entirely inside your browser. No image file, pixel data, OCR result, or contrast report is sent to any server at any point.

No accounts, no tracking, no analytics

There are no user accounts, no login, and no registration. No analytics scripts, tracking pixels, or telemetry are included. Your usage of this tool is not monitored or recorded.

No cookies

This tool does not set any cookies.

Local storage

One value is stored in your browser's localStorage: your chosen colour theme (light or dark). It is used only to restore your preference on your next visit. It is never read by any server.

Network requests

The only network requests this tool makes are a one-time download of the AI model files and the WebAssembly runtime it needs to analyse your images. These files are served from the same origin as the tool itself (image-colour-contrast-checker.timdixon.net) and are cached by your browser after the first download. No third-party servers or content delivery networks are involved.

The files downloaded on first use are:

  • PaddleOCR PP-OCRv4 text detection model (ch_PP-OCRv4_det_infer.onnx)
  • PaddleOCR PP-OCRv4 text recognition model (ch_PP-OCRv4_rec_infer.onnx)
  • PaddleOCR character key file (ppocr_keys_v1.txt)
  • ONNX Runtime Web WASM binaries

Total first-run download: approximately 28 MB. After that, your browser cache serves them instantly.

No third-party scripts

All JavaScript code is bundled and served from image-colour-contrast-checker.timdixon.net. No external scripts, fonts, or stylesheets are loaded from third-party origins.

Open source

The full source code is publicly available under the MIT Licence. You can read it, audit it, or run your own copy. See the GitHub repository.
